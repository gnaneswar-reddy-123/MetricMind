# MetricMind - Agentic Semantic BI Engine

## Phase 1: Database Design

This project focuses on designing the database layer for a Business Intelligence system.

## Database Design

The database follows a star schema approach.

### Dimension Tables:
- Customers
- Products
- Regions

### Fact Table:
- Sales Fact

## ER Diagram

The ER diagram represents relationships between dimension tables and the sales fact table using primary keys and foreign keys.

## Technologies Used

- MySQL
- MySQL Workbench
- SQL
